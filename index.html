AUDIO RECORDER

Beberapa Catatan & Cara menggunakan perekam suara :

  1. Tekan tombol yang bertulisan Start Recording untuk memulai perekaman suara
  2. Tekan tombol Stop Recording untuk menyelesaikan perekaman suara
  3. Perekeman dapat dilakukan secara berulang kali, dan hasil rekaman juga dapat di preview terlebih dahulu
  4. Setelah perekaman selesai, silahkan klik tombol kemudian klik tombol untuk mendownload file rekaman suara
