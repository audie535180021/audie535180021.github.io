AUDIO RECORDER

Beberapa Catatan & Cara menggunakan perekam suara :

  1. Tekan tombol yang bertulisan Start Recording untuk memulai perekaman suara
  2. Tekan tombol Stop Recording untuk menyelesaikan perekaman suara
  3. Perekeman dapat dilakukan secara berulang kali, dan hasil rekaman juga dapat di preview terlebih dahulu
  4. Harap Berikan Input Nama (untuk generate format nama file rekaman)
  5. Kalimat yang diucapkan :
    Perkenalkan Nama Saya _______ saat ini saya berusia ________ tahun
    Contohnya sebagai berikut :
  6. Setelah perekaman selesai, silahkan klik tombol untuk mendengarkan hasil rekaman suara
  7. Harap melakukan perekaman sebanyak 5x
  8. Setelah 5x menekan tombol stop recording, akan muncul button untuk mendownload semua file rekaman suara sekaligus
  9. Setelah mendownload file hasil rekaman, mohon bantuannya untuk mengisi dan mengupload file rekaman suara pada link google form berikut:
